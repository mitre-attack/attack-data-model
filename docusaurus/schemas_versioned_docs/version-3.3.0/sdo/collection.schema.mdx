## ObjectVersionReference

_Object containing the following properties:_

| Property                   | Description                                                                                                                      | Type                  |
| :------------------------- | :------------------------------------------------------------------------------------------------------------------------------- | :-------------------- |
| **`object_ref`** (\*)      | The ID of the referenced object.                                                                                                 | `string`              |
| **`object_modified`** (\*) | The modified time of the referenced object. It MUST be an exact match for the modified time of the STIX object being referenced. | `string` (_ISO 8601_) |

_(\*) Required._

## XMitreContents

Specifies the objects contained within the collection.

_Array of at least 1 [ObjectVersionReference](#objectversionreference) items._
